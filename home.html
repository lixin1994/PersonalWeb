Deployment Instruction

New instance in EC2

  • Create a new instance in EC2.
    1. Open AWS, select EC2 section, press launch instance.
    2. Select Ubuntu Server
    3. Choose any cpu you want
    4. Press next, then press next till Configure Security Group
    5. Configure the security like the table
      For HTTP the range should be anywhere
      For SSH the range should be your IP address
    6. Creat a SSH key or use an exist SSH key. If you create a SSH please download the SSH file usually is xxx.pem
